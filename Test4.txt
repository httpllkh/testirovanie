Баг-репорт
1. Название ошибки:
Буква в URL - NaN на счету

2. Ожидаемое поведение:
Игнорирование символа

3. Фактическое поведение:
При вводе любого символа кроме цифр в URL на счету появляется сумма NaN

4. Шаги для воспроизведения ошибки:
I. Открыть URL
II. Ввести любой символ в строку баланса
III. Нажать Enter

5. Операционная система:
Windows 10

6. Браузер:
Chrome 138

7. Частота возникновения ошибки:
Всегда

8. Влияние на работу системы:
Минимальное влияние, так как незачем вводить туда буквы

Баг-репорт
1. Название ошибки:
Точка и запятая

2. Ожидаемое поведение:
Точки и запятые можно использовать как разделитель для оплаты суммы с копейками

3. Фактическое поведение:
Точку и запятую невозможно вписать в поле для ввода суммы перевода

4. Шаги для воспроизведения ошибки:
I. Открыть URL
II. В поле "Сумма перевода" ввести любое число с точкой или запятой

5. Операционная система:
Windows 10

6. Браузер:
Chrome 138

7. Частота возникновения ошибки:
Всегда

8. Влияние на работу системы:
Серьезная ошибка, оплатить сумму вплоть до копеек не является возможным
